# Moto-Kart
MotoKard is an e-commerce website designed for buying and selling second-hand vehicles such as cars and bikes, along with vehicle spare parts and accessories. The platform allows motor shops and sellers to list their products and reach customers easily, similar to a marketplace model. Users can also compare prices from different sellers to find the most affordable and best deals
The UI/UX design for this project was created using Figma.
